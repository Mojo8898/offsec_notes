# git

Software development tracking

## Capabilities

```powershell
# Initialize a repo
git init $DIR

# Check repo status
git status

# Compare current with previous commits
git diff
git diff --staged

# Display branches
git branch

# Change branches
git switch $BRANCH

# Show commits within the current branch
git log

# Show changes related to a specified commit
git show $COMMIT_ID
```
